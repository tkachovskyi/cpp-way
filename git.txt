

Cd - переход в каталог(папку)
ls - посмотреть содержимое каталога(папки)
cat - просмотреть содержимое файла
git checkout -b git-txt  - создали локально(на компе) (ветку,бренчу"branch") с именем "git-txt"
git status - просмотр текущий статус репозитория.
git push - закинуть изменения на сервер.
git pull - стянуть(скачать) изменения с сервера.
git add . - добавить трекер на все(.) файлы в каталоге.
